// ── คำนวณเวลาคงเหลือก่อนเกษียณ (อายุราชการครบ 25 ปี รวมวันทวีคูณ) ──
function calcRetirement(joinDateStr, extraDays) {
  const joinDate = parseAnyDate(joinDateStr);
  if (!joinDate || isNaN(joinDate)) return '—';
  const retireMs   = joinDate.getTime() + (25 * 365.25 * 24 * 3600 * 1000) - ((extraDays || 0) * 24 * 3600 * 1000);
  const retireDate = new Date(retireMs);
  const now        = new Date();
  if (retireDate <= now) {
    return `<div style="background:linear-gradient(135deg,#1e6b1e,#2d8a2d);border-radius:8px;padding:8px 12px;text-align:center;box-shadow:0 2px 8px rgba(45,138,45,0.3);"><div style="font-size:13px;font-weight:700;color:#fff;">✅ ครบแล้ว</div></div>`;
  }
  const diffDays = Math.floor((retireDate - now) / (24 * 3600 * 1000));
  const ymd      = daysToYMD(diffDays);
  const _thM     = ['ม.ค.','ก.พ.','มี.ค.','เม.ย.','พ.ค.','มิ.ย.','ก.ค.','ส.ค.','ก.ย.','ต.ค.','พ.ย.','ธ.ค.'];
  const retireYear  = toBE(retireDate.getFullYear());
  const retireMonth = _thM[retireDate.getMonth()];
  const retireDay   = retireDate.getDate();
  const yearsLeft = diffDays / 365;
  let bgGrad, textColor, badgeColor;
  if (yearsLeft <= 1)      { bgGrad='linear-gradient(135deg,#c0392b,#e74c3c)'; textColor='#fff'; badgeColor='rgba(255,255,255,0.9)'; }
  else if (yearsLeft <= 3) { bgGrad='linear-gradient(135deg,#e67e22,#f39c12)'; textColor='#fff'; badgeColor='rgba(255,255,255,0.9)'; }
  else if (yearsLeft <= 7) { bgGrad='linear-gradient(135deg,#2980b9,#3498db)'; textColor='#fff'; badgeColor='rgba(255,255,255,0.9)'; }
  else { bgGrad='linear-gradient(135deg,#1a2332,#2a3f5c)'; textColor='var(--army-gold)'; badgeColor='rgba(200,168,75,0.9)'; }
  return `<div style="background:${bgGrad};border-radius:10px;padding:10px 14px;min-width:140px;box-shadow:0 3px 12px rgba(0,0,0,0.2);"><div style="font-size:11px;color:${badgeColor};font-weight:600;margin-bottom:4px;">⏱ อีก</div><div style="font-size:13px;font-weight:700;color:${textColor};line-height:1.3;">${ymd.text}</div><div style="display:inline-block;margin-top:5px;background:rgba(255,255,255,0.15);border:1px solid rgba(255,255,255,0.25);border-radius:6px;padding:2px 8px;font-size:11px;color:${badgeColor};font-weight:600;">${retireDay} ${retireMonth} ${retireYear}</div></div>`;
}

// ── คำนวณเวลาคงเหลือก่อนเกษียณอายุ 60 ปี (จากวันเกิด) ──
function calcRetirementAge60(birthDateStr) {
  const birthDate = parseAnyDate(birthDateStr);
  if (!birthDate || isNaN(birthDate)) return '—';
  const retireDate = new Date(birthDate.getFullYear() + 60, birthDate.getMonth(), birthDate.getDate());
  const now        = new Date();
  if (retireDate <= now) {
    return `<div style="background:linear-gradient(135deg,#1e6b1e,#2d8a2d);border-radius:8px;padding:8px 12px;text-align:center;box-shadow:0 2px 8px rgba(45,138,45,0.3);"><div style="font-size:13px;font-weight:700;color:#fff;">✅ เกษียณแล้ว</div></div>`;
  }
  const diffDays    = Math.floor((retireDate - now) / (24 * 3600 * 1000));
  const ymd         = daysToYMD(diffDays);
  const _thM        = ['ม.ค.','ก.พ.','มี.ค.','เม.ย.','พ.ค.','มิ.ย.','ก.ค.','ส.ค.','ก.ย.','ต.ค.','พ.ย.','ธ.ค.'];
  const retireYear  = toBE(retireDate.getFullYear());
  const retireMonth = _thM[retireDate.getMonth()];
  const retireDay   = retireDate.getDate();
  const yearsLeft = diffDays / 365;
  let bgGrad, textColor, badgeColor;
  if (yearsLeft <= 1)       { bgGrad='linear-gradient(135deg,#8e44ad,#9b59b6)'; textColor='#fff'; badgeColor='rgba(255,255,255,0.9)'; }
  else if (yearsLeft <= 5)  { bgGrad='linear-gradient(135deg,#16a085,#1abc9c)'; textColor='#fff'; badgeColor='rgba(255,255,255,0.9)'; }
  else if (yearsLeft <= 15) { bgGrad='linear-gradient(135deg,#2471a3,#2980b9)'; textColor='#fff'; badgeColor='rgba(255,255,255,0.9)'; }
  else { bgGrad='linear-gradient(135deg,#34495e,#4a6278)'; textColor='#e8f4fd'; badgeColor='rgba(232,244,253,0.85)'; }
  return `<div style="background:${bgGrad};border-radius:10px;padding:10px 14px;min-width:140px;box-shadow:0 3px 12px rgba(0,0,0,0.2);"><div style="font-size:11px;color:${badgeColor};font-weight:600;margin-bottom:4px;">⏱ อีก</div><div style="font-size:13px;font-weight:700;color:${textColor};line-height:1.3;">${ymd.text}</div><div style="display:inline-block;margin-top:5px;background:rgba(255,255,255,0.15);border:1px solid rgba(255,255,255,0.25);border-radius:6px;padding:2px 8px;font-size:11px;color:${badgeColor};font-weight:600;">${retireDay} ${retireMonth} ${retireYear}</div></div>`;
}
