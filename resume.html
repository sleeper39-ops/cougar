🔥 วิธีตั้งค่า Firebase (ทำครั้งเดียว)

2. สร้างโปรเจกต์ใหม่ → เลือก Realtime Database → Create database
3. เลือก Start in test mode (อ่าน/เขียนได้โดยไม่ต้อง login 30 วัน)
    หรือตั้ง Rules เป็น "read":true,"write":true ถาวร
4. คัดลอก Database URL รูปแบบ: https://xxx-default-rtdb.firebaseio.com
5. ไปที่ Project Settings → General → Web API Key → คัดลอก API Key

☁️ ตั้งค่า Google Drive สำหรับอัปโหลด
วิธีตั้งค่า Google Drive Upload:
1. ไปที่ Google Cloud Console
2. สร้าง Service Account → Download JSON key
3. Share โฟลเดอร์ Drive ให้ email ของ Service Account
4. หรือใช้ Google Drive API + OAuth token

⚠️ หมายเหตุ: การอัปโหลดไฟล์ไป Drive จาก browser โดยตรงต้องใช้ OAuth 2.0 — แนะนำให้ใช้ผ่าน Apps Script หรือ backend server แทน

ตอนนี้ระบบจะ บันทึกข้อมูลเอกสารลง Firebase และแสดงตัวอย่างไฟล์ในระบบไว้ก่อน
